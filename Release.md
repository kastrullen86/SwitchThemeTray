# SwitchThemeTray v1.0.0

Första stabila releasen av SwitchThemeTray.

Ett lättviktigt Windows system tray-verktyg för att snabbt växla mellan Light och Dark theme.

## Funktioner
- Växla mellan Light / Dark theme
- Körs tyst i system tray
- Ingen installation krävs
- Samma funktionalitet i EXE, BAT och PS1

## Filer i releasen
- SwitchThemeTray.exe (rekommenderad)
- SwitchThemeTray.ps1
- SwitchThemeTray.bat

## Notering
EXE-filen är endast en paketering av PowerShell-scriptet – ingen extra logik.
